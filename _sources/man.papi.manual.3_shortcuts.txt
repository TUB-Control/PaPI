Shortcuts
============

In PaPI there are some easy hotkey's defined to enable navigation and work using keyboard only.



.. list-table:: Implemented key sequences
    :widths: 10 10 8
    :header-rows: 1

    * - Key squence
      - Action
      - Where to use
    * - CTRL+L
      - Load Dialog
      - Main Window
    * - CTRL+S
      - Save Dialog
      - Main Window
    * - CTRL+N
      - Plugin Create Menu
      - Main Window
    * - CTRL+O
      - Plugin Overview Menu
      - Main Window
    * - CTRL+A
      - Select previous tab
      - Main Window
    * - CTRL+D
      - Select next tab
      - Main Window
    * - Arrow Up/Down
      - Move in tree
      - Overview/Create Menu
    * - Enter/Return
      - Set focus: search to tree
      - Overview/Create Menu
