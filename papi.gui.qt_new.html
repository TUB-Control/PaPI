class papi.gui.qt_new.overview_menu.OverviewPluginMenu(gui_api, parent=None)[source]¶

Bases: PySide.QtGui.QMainWindow, papi.ui.gui.qt_new.overview.Ui_Overview

This class is used to create an extra window which is used to display all created plugins. The information are taken by the corresponding DPlugin-Object of a plugin. By this window a user is able to create and cancel subscriptions.
