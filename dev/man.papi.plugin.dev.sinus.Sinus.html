Sinus¶

General description

This plugin is used as simple data source of different sine signals.

Configuration¶

The plugin uses this specific configuration.

Plugin configuration
Name Type Description
amax Int Name of the signal whose first value is used to set the current value for the progress bar.
f Float Frequency, currently not used.

Parameter¶

A plugin instance can be manipulated by the following parameter.

Provided parameter
Name Type Example Description
Frequenz Block SinMit_f3 Float 0.3 Sets the frequency of the sine signal f3.
